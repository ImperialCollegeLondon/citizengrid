Lets start with a quick overview of the CitizenGrid platform...

Application Directory

The application directory shows all the applications that are available to you. To run an application, select it from the directory to see further information and details of how to run the application or contribute your spare computing power to the project.

Want to register an application?

If you have an application you want to register with CitizenGrid, select the application provider role below.

Account Profile and Settings

Use your account profile to provide details that identify you or your project to other CitizenGrid users and to configure your cloud credentials to contribute cloud computing time to projects.

Run locally or on a cloud

The application you select may be run on one of your local machines or it may be run on cloud resources that you provide. You can configure remote cloud platforms in your account settings.

To get started, select your primary role from the options on the right (you can switch between roles later).
